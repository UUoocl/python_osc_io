OSC IO Browser Source¶

OSC IO: Browser Source Integration¶

This script provides an OSC (Open Sound Control) interface for OBS Studio, allowing incoming OSC messages to trigger JavaScript events within Browser Sources. It also supports sending OSC messages when a Text Source is updated.

Co-created with Google AI Studio.

OSC IO Text Source¶

OSC IO: Text Source Integration¶

This script provides an OSC (Open Sound Control) interface for OBS Studio, focusing on updating OBS Text Sources with received OSC data. It also supports sending OSC messages when a mapped Text Source is updated.

Co-created with Google AI Studio.
